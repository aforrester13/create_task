      Welcome to the GPA Calculator!

      Quick Tips:
    • Add classes before clicking the calculate button. Adding classes after clicking the calculate button will not properly calculate your grade point average. If you wish to add more classes, refresh the page and then add classes BEFORE pressing the calculate button.

    • If you add a class, input the correct value for the grade and number of hours. You cannot change the value of the grade or number of hours after it is inputted without refreshing the page.

    • Good Luck!
